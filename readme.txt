Comptes Manon — Application Web
=================================

1️⃣ Installation
----------------
- Clonez ou téléchargez le dossier sur votre ordinateur.
- Hébergez-le sur GitHub Pages (Settings > Pages > Deploy from branch).
- L'application sera accessible via une URL publique GitHub.

2️⃣ Connexion Google Sheets et Drive
------------------------------------
- Créez 3 Google Sheets : CompteChèque, Livret, PrévisionnelChèque.
- Créez un script Google Apps Script (Extensions > Apps Script) pour gérer les lectures/écritures.
- Déployez-le en tant que WebApp et copiez l’URL.
- Modifiez app.js pour appeler cette URL via fetch().

3️⃣ Fonctionnalités
--------------------
- Lecture/écriture des données Google Sheets
- Upload d'image vers Google Drive
- Calcul du solde prévisionnel
- Export CSV
- PWA installable (iPhone, Android)

4️⃣ Design
-----------
- Couleur principale : #BD33A4
- Animation douce (Fade + Slide)
- Interface responsive et fluide
